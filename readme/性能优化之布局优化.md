## 5-1 Android绘制原理及工具选择
    绘制原理
        CPU负责计算显示内容
        GPU负责栅格化（UI元素绘制到屏幕上）
        16ms发出VSync信号出发UI渲染
        大多数的Android设备屏幕刷新频率：60hz
    优化工具
        Systrace
            .关注Frames
            .正常：绿色圆点，丢帧：黄色或红色
            .Alerts栏
        Layout Inspector
        
        Choreographer
            获取FPS,线上使用，具备实时性
            .Api 16之后
            .Choreographer.getInstance().postFrameCallback
